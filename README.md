# Better-Life

習慣養成與任務達成類型的App，使用者可記錄每日任務執行情況與觀看任務執行成果，並與其他使用者共同完成任務。

## 使用技術與工具

- 設計模式：MVVM、Singleton
- Android Jetpack - ViewModel、LiveData、Lifecycle、Data Binding、Navigation
- 第三方套件：Glide、ImagePicker、Lottie
- 分析工具：Firebase Crashlytics


## 開發環境
- Android Studio 4.1.2
- Android SDK 30
- Gradle 4.0.1
