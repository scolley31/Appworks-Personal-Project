# Better Life

習慣養成與任務達成類型的App，使用者可記錄每日任務、觀看任務執行成果，並與其他使用者共同完成任務。

[<img width="215" height="83" src="https://firebasestorage.googleapis.com/v0/b/triptrip-5f1ab.appspot.com/o/google-play-badge.png?alt=media&token=0d0f7868-0e2c-49ab-bc1d-33e70daa6bfd"/>](https://play.google.com/store/apps/details?id=com.scolley.betterlife)

## 功能介紹
### 首頁
+ 使用者可瀏覽個人的每日任務執行百分比
+ 使用者可確認逐項個人任務與雙人任務的進度

<img src="https://github.com/scolley31/Better-Life/blob/main/mockup%20(2).png" width="25%" height="25%"> </a>
<img src="https://github.com/scolley31/Better-Life/blob/main/mockup%20(13).png" width="25%" height="25%">

### 探索任務頁
+ 使用者可瀏覽其它使用者發布的任務，並可將此任務加入到自己的個人任務中

<img src="https://github.com/scolley31/Better-Life/blob/main/mockup%20(6).png" width="25%" height="25%">

### 新增任務頁
+ 使用者可新增個人任務，並設定名稱、種類與任務達成方式

<img src="https://github.com/scolley31/Better-Life/blob/main/mockup%20(5).png" width="25%" height="25%">

### 任務執行頁
+ 使用者可計時完成每日任務

<img src="https://github.com/scolley31/Better-Life/blob/main/mockup%20(4).png" width="25%" height="25%">

### 任務詳細頁
+ 使用者可觀看每個任務的執行成果

## 使用技術與工具

- 設計模式：MVVM、Singleton
- Android Jetpack - ViewModel、LiveData、Lifecycle、Data Binding、Navigation
- 第三方套件：Glide、ImagePicker、Lottie
- 分析工具：Firebase Crashlytics


## 開發環境
- Android Studio 4.1.2
- Android SDK 30
- Gradle 4.0.1
